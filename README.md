  - `git clone https://github.com/Hsiao1020/model-api-v0.git`

    - Windows - `.\\venv\Scripts\activate`
  - `pip install -r requirements.txt`
 
- **執行**:
  - macOS- `source venv/bin/activate`  
  - Windows - `.\\venv\Scripts\activate`
  - `flask run`
